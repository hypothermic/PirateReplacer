# Pirate Replacer

Replaces all links to `thepiratebay.org` with their Tor Onion link (`uj3wazyk5u4hnvtk.onion`).

### Other domains

Not only the `.org` domain gets replaced, but also the following:

`thepiratebay.se`
`thepiratebay.gs`
`thepiratebay.la`
`thepiratebay.vg`
`thepiratebay.mn`
`thepiratebay.gd`
`thepiratebay.am`
